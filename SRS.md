# 프로젝트 개요

## 1. 기획의도

- 숏폼, 다양한 업무용 툴, 비대면 근무 환경으로 인한 집중력 관리 및 작업 이력 도구의 필요성 대두
- 타이머와 업무일지, 회고록 등을 통합하여 생산성 있는 하루의 흐름을 유기적으로 기록하고 돌아볼 수 있는 서비스

## 2. 목표

- `집중력 향상` : 포모도로 기법 기반 25분 집중 / 5분 휴식 사이클 자동화
- `작업 이력 관리` : 포모도로 완료 시 DailyLog 자동 연동으로 기록 습관 형성
- `회고 문화 정착` : 주제별 템플릿 제공으로 손쉬운 회고 작성
- `몰입 환경 제공` : BGM 믹서, 배경화면 테마, 풀스크린 집중 모드
- `맞춤형 통계` : 일/주/월 집중시간 차트 및 GitHub 잔디 스타일 히트맵

## 3. 사용대상

집중력 관리와 작업 이력 기록이 필요한 개발자 및 지식 근로자

- 하루 작업을 체계적으로 기록하고 싶은 개발자
- 포모도로 기법으로 업무 집중도를 높이고 싶은 사용자
- 과거 작업 이력을 달력·통계로 한눈에 파악하고 싶은 사용자

## 4. 필수 구현 기능

### 4-1. 포모도로 타이머

- 토마토 SVG 진행 바
- 인터벌 단위
    - ex.`집중(25분) / 단기휴식(5분) / 장휴식(15~30분) 모드 전환`
    - 카운트다운 / 일시정지 / 재시작 / 4포모도로 완료 시 장휴식 자동 전환
    - 🍅×4 세션 카운터 시각화
- 유저 인터렉션
    - 완료 사운드 - ex.`케찹짜는 소리` (Web Audio API)
    - 브라우저 Toast/Push 알림 (Notification API)

### 4-2. 투두리스트

- 과거날짜 태스크 중 미완료 태스크 → 다음 날 자동 이월 (ON/OFF 설정)
- 드래그앤 드랍(우선순위)
- 메인화면에 노출

### 4-3. 데일리로그

- 마크다운 에디터 기반 일일 작업 기록 (react-md-editor)
- 자동 저장 여부 / 수정 / 삭제

### 4-4. 회고로그

- 4종 회고 템플릿 선택 : `Tech` / `Decision` / `Communication` / `Emotion`
- DailyLog 와 연동

### 4-5. 몰입환경

- BGM 플레이어 : Lo-fi / 빗소리 / 카페 선택 및 볼륨 슬라이더 - suno 사용
- 배경화면 갤러리 5종 선택 저장 (캠프파이어 / 비오는 카페 / 새벽 코딩 등) - dotart
- 풀스크린 집중 모드 (ESC 해제, BGM 유지)

### 4-6. 달력 & 대시보드

- 날짜별 집중시간 GitHub 잔디 스타일 히트맵
- 달력 날짜 클릭 → DailyLog + 회고 통합 상세 페이지

### 4-7. 인증

- JWT Access / Refresh 토큰 발급 및 Rotation
- Supabase RLS(Row Level Security) 사용자별 데이터 격리
